# Medidas de Imagens — Site Phytonatus

Guia completo para envio de imagens pela Ana. Todas as medidas já estão em **2x (retina)** para qualidade em telas HiDPI — mas pode enviar maior sem problema.

- Formato preferido: **JPG** para fotos (fundo/produto) · **PNG transparente** para logos/selos · **MP4/WebM** para vídeos
- Compressão: usar [tinypng.com](https://tinypng.com) ou [squoosh.app](https://squoosh.app) antes de enviar
- Peso máximo ideal: fotos hero até 500 KB · fotos secundárias até 250 KB · logos até 100 KB
- **Importante:** em toda imagem `cover` (ocupa área sem distorcer), o elemento principal deve ficar **centralizado**, com margem de segurança nas bordas — as laterais podem ser cortadas em telas mais estreitas
